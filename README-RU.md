## Описание  
  
Невероятный дискорд бот по имени Самурай. Он просто незаменим на любом сервере. Самурай может создавать систему уровней, модерировать участников, поздравлять их с днем рождения. Также самурай может играть в простые игры (крестики нолики и четыре в ряд) с разными тактиками. Может проигрывать композиции с ютуб, создавать очереди и плейлисты. Поразительно точно реагирует на обращения к нему и всегда поддержит беседу. Рекомендуется к установке на каждый сервер.
  
## Функции  
  
Самурай изо всех старается быть живым. Вот список его способностей с командами discord. Используйте команды с префиксом **.** Пример: **.help**.
  
> **help** - Основная команда. Выводит описание и список команд бота.  
 
### Глоссарий
>Модуль Самурая отвечающий за языки и настроение.
Позволяет задать необходимый язык или настроению Самураю.

| command | description |  
| --- | --- |
| **gl_help** | Выводит помощь по глоссарию |
| **gl** | Выводит текущие настройки бота |
| **gl_lang \<lang>** | Задает выбранный язык боту. **Use abbreviations** |
| **gl_vibe \<vibe>** | Задает настроение боту |


### Модуль Музыка  
> Модуль Самурая, отвечающий за музыку. Позволяет создавать плейлисты, слушать, ставить на паузу ролики с [youtube](https://www.youtube.com/).  
Отлично подходит для посиделок с друзьями или поможет настроиться на нужную волну.  
  
| command | description |  
| --- | --- |  
| **player_help** | Выводит помощь по музыкальному модулю |  
| **join** | Присоединяет бота к вашему голосовому каналу |  
| **leave** | Отключает вашего бота от голосового канала и очищает очередь |  
| **play \<url** or **keywords>** | Добавляет заданные данные в очередь, если очередь была пуста, начнет воспроизведение  |
| **pause** | Останавливает воспроизведение |
| **resume** | Возобновляет воспроизведение |
| **restart** | Играет песню занаво |
| **stop** | Останавливает воспроизведение и очищает очередь |
| **skip** | Пропускает текущую песню |
| **skipto \<index>** | Переходит в выбранной песне |
| **previous** | Возвращается к предыдущей песне |
| **volume \<value>** | Задает уровень громкости |
| **volume up** | Повышает громкость на 10% |
| **volume down** | Понижает громкость на 10% |
| **lyrics** | Выводит текст песни |
| **queue** | Выводит текущую очередь |
| **shuffle** | Перемешивает очередь |
| **loop (loop all)** | Зацикливает очередь |
| **loop 1** | Зацикливает песню |
| **loop none** | Останавливает зацикливание |
| **eq [flat, metal, boost, piano]** | Задает эквалайзеру выбранные настройки |
| **adveq \<band> \<gain>** | Задает выбранной частоте уровень громкости |
| **np** | Выводит информацию о текущем треке |
| **seek <time>** | Пропускает трек до выбранного времени | 
  
### Модуль Уровни  
> Модуль Самурая, отвечающий за систему уровней на сервере. Позволяет создавать конкурентную среду и систему прокачки в вашей компании.  
  
| command | description |  
| --- | --- |  
|**level_help** | Выводит помощь по командам, связанных с уровнями |  
|**level_add \<role> \<xp>** | Создаст уровень с ***role*** и ***xp*** для его получения |  
|**level_update \<role> \<xp>** | Обновит уровень ***role*** на выбранное количество ***xp*** |  
|**level_delete** | Удалит роль из базы данных |  
|**level_show** | Выведет список всех уровней сервера и количество опыта для их получения |  
|**level** | Выведет эмбед о вашем уровне |  
|**level_dashboard** | Выведет топ участников сервера |  
  
### Модуль Дни Рождения  
> Модуль Самурая, отвечающий за дни рождения.  
Заводит базу данных дней рождений участников, куда каждый желающий может внести свои данные. Бот поздравит участника в этот замечательный день.  
  
| command | description |  
| --- | --- |  
| **bd_help** | Выведет помощь по командам, связанных с днем рождения. |  
|**bd_set_chat <chat>** | Задаст чат куда будут приходить поздравления |  
|**bd_up_chat <chat>** | Обновит чат для поздравлений |  
|**bd_del_chat** | Удалит чат для поздравлений, **вы не будете получать поздравления Самурая** |  
|**bd_show_chat** | Выведет чат, куда будут поступать поздравления |  
| **bd_add \<year> \<month> \<day>** | Инициализирует нового участника в базу данных с указанной датой |  
| **bd_update \<year> \<month> \<day>** | Обновит дату рождения |  
| **bd_delete**| Удалит дату рождения |  
| **bd** | Выведет эмбед о вас |  
| **bd_show** | Выведет список всех инициализированных пользователей с датами дней рождений |  
  
### Модуль Крестики-нолики  

---
Не работает на данном релизе

> Модуль Самурая, отвечающий за "крестики-нолики".  
Позволяет участникам играть в крестики-нолики. Участники могут соревноваться между собой или попытать свои силы против самурая, с адаптивным уровнем сложности игры и разными типами стратегий.  
Поможет решить споры или потренировать мозг.  
  
| command | description |  
| --- | --- |  
| **xo_help** | Выводит помощь по командам, связанных с крестиками-ноликами |  
| **xo_rules** | Выводит правила игры в крестики-нолики |  
| **xo \<member1> \<member2>** | Инициализирует новую игру для ***member1*** и ***member2*** |  
| **xo \<member1> \<Samurai> <game type>** | Инициализирует новую игру для ***member1*** и ***Самурая*** со стратегией бота ***game type*** (game types: "*ai*", "*random*", **default game type** = "*ai*") |  
| **xo_place \<number>** | Поставит в клетку ***number*** нужную фишку |  
| **xo_lose** | Текущий игрок сдастся |

### Модуль Переводчик  
> Модуль Самурая, отвечающий за иностранные языки.  
Всегда под рукой, необходимо что-то перевести — Самурай тут как тут, не знаешь на каком языке написанно сообщение — Самурай всегда поможет. Когда скучно, можно попытаться угадать язык сообщения. В общем чудо, а не модуль!  
  
| command | description |  
| --- | --- |  
| **tr_help** | Выводит помощь по командам, связанных с модулем переводчик |  
| **tr_list** | Выведет список кратких обозначений языков  |  
| **tr_trans \<source lang> \<target lang> \<message>** | Переведет ***message*** с ***source lang*** на ***target lang*** |  
| **tr_trans \<target lang> \<message>** | Определит язык исходного сообщения и переведет ***message*** на ***target lang*** |  
| **tr_detect_lang \<message>** | Выводит язык ***message*** |  
| **tr_game** | Начнет игру ***"угадай язык"*** |
